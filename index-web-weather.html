Weather 5-Day Forecast

The purpose of this project was to create a weather dashboard that shows a 5-Day forecast. Please see below for the fully deployed 5-Day Weather Forecast application. It uses a weather API that I got here. This was to help us learn how to use a Server-Side API with a functioning weather dashboard.

the challenge

I began by doing research and finding sources -- these videos were equally helpful in various parts of the assignment.

Sources: Weather API Forecast Resource 1, Weather API Forecast Resource 2, Coding Source


I followed the resources to create the Javascript, HTML, and CSS. Here are some screenshots of how my process worked. First, I grabbed colors I liked more and messed around with them for the weather palette. These were adjusted slightly as I went along to make something functioning and cool, but still kept the colors in a teal-palette I liked.

Then, I got to work making the HTML, CSS, JavaScript, and weather API files. Below, you can see some screenshots of the web application in process and how the weather API functions as a whole.

When the page first loads, the user's location automatically pops up as seen in the left screenshot. We were to use local storage to save the recent searches and make them linkable, as you can see with the selected button.

Here, you can see the page's responsiveness on a mobile device where the app functions on a smaller screen. It uses weather icons that update accordingly to the weather report, as well as the date, temperature, wind and humidity. For extra features, the search function searchable by city, state, just by city, or by zip code, and I used localStorage for saving the recent search history.

In conclusion

In conclusion, I set up the quiz to meet the challenge requirements and improved it. I:

  • - provided a weather dashboard with form inputs and the user has the ability to search for any given city that will show up in the results

  • - you can find the current and future conditions for that city, and it's added to the search history off to the left side of the page, and also easily check your current location

    • - the user can view future weather conditions for that city in a 5-day forecast that includes a weather icon, the temperature, wind, and humidity

    - you can click on a city in the search history and that city is pulled up, where the user is then shown the recent city
