Weather 5-Day Forecast

The purpose of this project was to create a weather dashboard that shows a 5-Day forecast. Please see below for the fully deployed 5-Day Weather Forecast application. It uses a weather API that I got here. This was to help us learn how to use a Server-Side API with a functioning weather dashboard.

I began by doing research and finding sources -- these videos were equally helpful in various parts of the assignment.
